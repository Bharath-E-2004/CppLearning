# ЁЯФ╖ What is a Class?

A **class** is a blueprint (template).
It defines:

* Variables (data members)
* Functions (member functions)

Think of it like a design of a house ЁЯПа
But not the real house.

---

# ЁЯФ╖ What is an Object?

An **object** is a real instance of a class.

If class = blueprint
Object = actual house built from it.

---

# ЁЯФе Basic Syntax

```cpp
class ClassName {
public:
    // data members
    // member functions
};
```

# ЁЯФе Simple Example

```cpp
#include <iostream>
using namespace std;

class Student {
public:
    string name;
    int age;

    void display() {
        cout << name << " " << age;
    }
};

int main() {
    Student s1;   // object creation

    s1.name = "Bharath";
    s1.age = 20;

    s1.display();
}
```

Output:

```

```

# ЁЯФ╖ Access Specifiers (VERY IMPORTANT)

C++ has 3:

1. `public` тЖТ accessible everywhere
2. `private` тЖТ accessible only inside class
3. `protected` тЖТ used in inheritance

Example:

```cpp
class Student {
private:
    int marks;

public:
    void setMarks(int m) {
        marks = m;
    }

    int getMarks() {
        return marks;
    }
};
```

Now you cannot do:

```cpp
s1.marks = 90;  // тЭМ Error
```

You must use:

```cpp
s1.setMarks(90);
```

ЁЯСЙ This is called **Encapsulation**.

# ЁЯФе Constructor (Important Topic)

A constructor is a special function:

* Same name as class
* Runs automatically when object is created

```cpp
class Student {
public:
    string name;

    Student(string n) {   // constructor
        name = n;
    }

    void display() {
        cout << name;
    }
};

int main() {
    Student s1("Bharath");
    s1.display();
}
```

Output:

```
Bharath
```
