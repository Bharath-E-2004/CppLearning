🔹 if – else if – else (Ladder)

Used when you have multiple conditions.

int marks = 75;

if (marks >= 90) {
    cout << "Grade A";
}
else if (marks >= 75) {
    cout << "Grade B";
}
else if (marks >= 50) {
    cout << "Grade C";
}
else {
    cout << "Fail";
}

🧠 Comma Operator Behavior

In C++, the comma inside condition is called the comma operator.

It evaluates both expressions, but only the last expression decides the result.

Example:

int a = 1;
int b = 1;

while (a < 5, b < 5) {
    cout << b << " ";
    b++;
}
