# 🔹 What is a Pointer?

👉 A pointer stores the **address of another variable**.

Example:

```cpp
int a = 10;
int *p = &a;
```

Here:

* `a` → value = 10
* `&a` → address of `a`
* `p` → stores address of `a`

---

# 🔥 Basic Example

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int *p = &a;

    cout << a << endl;     // 10
    cout << &a << endl;    // address
    cout << p << endl;     // address
    cout << *p << endl;    // 10
}
```

---

# 🔹 Important Symbols

| Symbol | Meaning              |
| ------ | -------------------- |
| `&`    | Address-of operator  |
| `*`    | Dereference operator |

---

### 🧠 What does `*p` mean?

If:

```cpp
int *p = &a;
```

Then:

```cpp
*p
```

means “go to the address stored in p and get the value”.

---

# 🔥 Modifying Value Through Pointer

```cpp
int a = 10;
int *p = &a;

*p = 50;

cout << a;   // 50
```

Changing via pointer changes original variable.

---

# 🔥 Pointer and Array (VERY IMPORTANT)

```cpp
int arr[3] = {10, 20, 30};

cout << arr;        // address of first element
cout << &arr[0];    // same
```

👉 Array name = pointer to first element.

You can do:

```cpp
int *p = arr;

cout << *(p + 1);   // 20
```

---

# 🔥 Null Pointer

```cpp
int *p = nullptr;
```

Good practice to avoid garbage pointer.

---

# 🔥 Dynamic Memory (Heap)

```cpp
int *p = new int(100);
cout << *p;
delete p;
```

* `new` → allocate memory
* `delete` → free memory

If you forget `delete` → memory leak.

---

# 🔥 Pointer to Pointer

```cpp
int a = 10;
int *p = &a;
int **pp = &p;
```

* `*p` → 10
* `**pp` → 10

---

# 🚨 Dangerous Example

```cpp
int *p;
cout << *p;   // ❌ undefined behavior
```

Uninitialized pointer = danger zone.
